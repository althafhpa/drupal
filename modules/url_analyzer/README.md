# URL Analyzer for Drupal 10

A powerful module to collect, analyze and export all URLs from your Drupal site.

## Features

- Collects URLs from multiple sources:
  - Core paths and aliases
  - Entity canonical URLs
  - Taxonomy terms
  - Menu links
  - Media entities
  - User profiles
  - Webforms
  - Commerce products
  - Search pages
  - Custom blocks
  - Link fields
  - Views paths

- Admin Interface
  - Paginated URL listing
  - Real-time URL filtering
  - Bulk selection capabilities
  - Export functionality (JSON/CSV)
